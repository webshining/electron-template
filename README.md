## Getting started
<hr>

### Init Project:
```bash
git clone https://github.com/webshining/electron-template project-name
cd project-name
npm i
# Write npm i --platform=win32 for start from wsl in windows
```
### Start Template:
```bash
npm start
```
### Start Test:
```bash
npm run test
```
